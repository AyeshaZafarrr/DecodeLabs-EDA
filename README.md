# Exploratory Data Analysis (EDA)

## Project Overview
This project analyzes an e-commerce dataset using Microsoft Excel. It includes summary statistics, PivotTables, charts, and business insights to understand sales performance and customer behavior.

## Tools Used
- Microsoft Excel

## Analysis Performed
- Summary Statistics
- PivotTables
- Product vs Total Sales Chart
- Order Status Distribution (Pie Chart)
- Referral Source vs Number of Orders
- Histogram / Distribution Analysis
- Business Insights
